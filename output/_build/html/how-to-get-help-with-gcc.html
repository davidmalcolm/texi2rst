How To Get Help with GCCÂ¶

If you need help installing, using or changing GCC, there are two ways to find it:

  • Send a message to a suitable network mailing list. First try gcc-help@gcc.gnu.org (for help installing or using GCC), and if that brings no response, try gcc@gcc.gnu.org. For help changing GCC, ask gcc@gcc.gnu.org. If you think you have found a bug in GCC, please report it following the instructions at see How and Where to Report Bugs.
  • Look in the service directory for someone who might help you for a fee. The service directory is found at http://www.fsf.org/resources/service.

For further information, see http://gcc.gnu.org/faq.html#support.
